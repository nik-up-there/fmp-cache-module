## Features included in the module

Hi, if you want to add a Redis cache within your NestJS backend, you are at the right place. <br />
Redis is in-memory data structure store that can be used as a cache or has a message broker with events. <br />
In this module, we will use one master redis node, without any replica.


In this module, I will explain all the steps that you need in:
- your NestJS backend
- the docker setup
